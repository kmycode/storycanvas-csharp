◆ OverwriteOption

enum OneDriveRestAPI.Model.OverwriteOption
strong

OverwriteOption is used for upload load operations. It specifies what should be done in case of a file name conflict.

列挙値
DoNotOverwrite 

Does not perform an overwrite if a file with the same name already exists.

Overwrite 

Overwrites the exisiting file with the new file's contents.

Rename 

If a file with the same name exists, this option will rename the new file.
